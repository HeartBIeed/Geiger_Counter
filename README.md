Проект счетчика Гейгера на 8 бит MCU ATmega16.

За основу взял схему из статьи: https://nice.artip.ru/shema-schyotchika-geygera 

В качестве генератора используется выход ШИМ с микроконтроллера. 

555 таймер удлиняет выход со счетчика Гейгера и сигнал выводится на светодиод, зуммер и на вход внешнего прерывания MCU.

Контроллер непрерывно считает количество событий за последние 40 секунд и выводит это число на LCD 1602 дисплей (параллельный порт 4 бит).

DS1307 - часы реального времени. I2С интерфейс.
